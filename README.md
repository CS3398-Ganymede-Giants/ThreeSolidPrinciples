### Ant Build Options
- Clean, build, and run
```shell
$ ant main
```
- Clean and build
```shell
$ ant clean-build
```
- Clean
```shell
$ ant clean
```
- Compile classes
```shell
$ ant compile
```
- Build jar files
```shell
$ ant jar
```
- Run program
```shell
$ ant run
```
